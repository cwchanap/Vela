# Mobile MVP M1: Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create `apps/vela-mobile` as an independent Quasar + Capacitor iOS application shell that builds and runs in the Vela monorepo.

**Architecture:** A standalone Quasar v2.4 app-vite workspace using Quasar's built-in Capacitor mode. Mobile-owned routing, layouts, and a 5-tab bottom navigation stub. No shared package dependencies — `@vela/common` is deferred to M1's continuation (its barrel imports `@tanstack/vue-query`).

**Tech Stack:** Quasar v2.16 + `@quasar/app-vite` v2.4, Vue 3.4, Vue Router 4.3, Capacitor 7, TypeScript 5.6, Vitest 3.2, Bun 1.3.1+

**Spec:** `docs/superpowers/specs/2026-07-12-mobile-mvp-m1-scaffold-design.md`

## Global Constraints

- **Bun only.** Use `bunx` instead of `npx` for all CLI invocations.
- **No `@vela/common` dependency.** Its barrel re-exports `config.ts` which imports `@tanstack/vue-query`. Do not add it to dependencies or alias it.
- **Capacitor config is JSON**, not TypeScript. Quasar app-vite v2.4 template ships `capacitor.config.json`.
- **Router mode** uses `process.env.VUE_ROUTER_MODE` (auto-switches to hash for Capacitor). Never hard-code `createWebHistory`.
- **Xcode 16+** required (Capacitor 7 requirement). Dev machine has Xcode 26.6.
- **CocoaPods** must be installed before iOS sync (`brew install cocoapods`). Not currently installed.
- **`productName: "Vela"`** in package.json — Quasar derives the native app name from it.
- **Root-level files touched:** `eslint.config.js`, `turbo.json`, `package.json`, `bun.lock`

```ts
import { defineRouter } from '#q-app/wrappers';
import {
  createMemoryHistory,
  createRouter,
  createWebHashHistory,
  createWebHistory,
} from 'vue-router';
import routes from './routes';

export default defineRouter(() => {
  const createHistory = process.env.SERVER
    ? createMemoryHistory
    : process.env.VUE_ROUTER_MODE === 'history'
      ? createWebHistory
      : createWebHashHistory;

  const Router = createRouter({
    scrollBehavior: () => ({ left: 0, top: 0 }),
    routes,
    history: createHistory(process.env.VUE_ROUTER_BASE),
  });

  return Router;
});
```
